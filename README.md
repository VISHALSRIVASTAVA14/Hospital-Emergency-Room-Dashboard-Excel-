# Hospital-Emergency-Room-Dashboard-Excel-
This project showcases an interactive and data-driven Emergency Room (ER) Dashboard created using Microsoft Excel. The dashboard provides a clear and visual overview of hospital performance metrics related to patient flow, waiting times, satisfaction, and departmental referrals.

1) **Monthly Patient Count (2023 & 2024)**:
Visual breakdown of total patients for each month.

2) **Average Waiting Time**:
Displays the mean time patients spent waiting before receiving care.

3) **Patient Satisfaction Score**:
Highlighting the overall service quality through satisfaction metrics.

4) **Gender-Based Analysis**:
Easy-to-read charts comparing patient distribution across genders.

5) **Department Referrals**:
Shows how many patients were referred to each department every month.

6) **On-Time vs Delayed Service**:
Comparison chart showing how many patients were served on time vs delayed.
